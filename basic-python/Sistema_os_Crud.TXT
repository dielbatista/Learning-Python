Entendendo a Estrutua do Sistema_OS_CRUD

├── app.py                # Ponto de entrada (Interface Streamlit e Navegação)
o arquivo app.py define as chamadas de funções na aplicação.

tais como a funação def show_main_app()
    que define e mostra a aplicação na web 
e a funcao de def show_login_page() 
    que define as configurações de login de cada usuario(técnico logado ou tentando logar ao sistema)
├── auth.py               # Lógica de Login, Logout e verificação de Sessão
aplica a lógica e define funcoes de validação de login e Logout
    tais como a função de que gera um hash_password
    e funções de verificação de password e validalção de password
    são as 3 principais funções de validação de uma senha.
├── database.py           # Conexão com Banco (Engine) e criação de tabelas
    Aqui é centralizado a criação do engine. se for trocado o banco de dados de Ex:(SQLite para MySQL) apenas o arquivo database.py é modificado
├── requirements.txt      # Lista de bibliotecas (streamlit, sqlalchemy, etc.)
    define as dependencias requeridas para que o sistema funcione.

    importante detalhe na criação de senha, definir hashes de senhas, pois caso aconteça roubo de banco de dados
    não é possivel verificar as senhas reais
