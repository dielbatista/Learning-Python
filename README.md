# 🚀 Back-end Study Roadmap with Python

This repository was created with the goal of **organizing and guiding my Back-end studies using Python**, from the fundamentals to more advanced concepts used in the market.

It can also serve as a reference for anyone who is starting or structuring their studies in the area.

---

## 🎯 Objective

* Learn the fundamentals of **Back-end development**
* Master **Python** applied to APIs and services
* Understand best practices, architecture, and testing
* Build a solid foundation for professional work

---

## 🧠 Study Topics

### 🔹 Programming Fundamentals

* Basic Python syntax
* Data types
* Conditional structures (`if / else`)
* Loops (`for`, `while`)
* Functions

### 🔹 Object-Oriented Programming (OOP)

* Classes and objects
* Encapsulation
* Inheritance
* Polymorphism

### 🔹 Back-end with Python

* File handling
* Error and exception handling
* Virtual environments
* Dependency management

### 🔹 APIs and Web

* HTTP (requests and responses)
* REST APIs
* Frameworks (Flask / FastAPI / Django)
* Authentication and authorization

### 🔹 Databases

* Basic SQL
* Python database integration
* ORM
* Relational database concepts

### 🔹 Testing

* Unit testing
* Testing best practices
* Python testing tools
